SAM AudioやSAM3を活用し、以下の動作をするアプリを作りたい。必要に応じてその他モデルの調査・適用もして。

# 機能概要
- 入力：動画（音声付き）
- 対象物をクリックすると、対象物がセグメンテーションされる
    - 動画の状態でセグメントされるとBest
- クリックされた対象が何なのか説明文を生成する
    - API、ローカルモデルなんでもOK
- 生成文をもとに、SAM Audioで対象物を指定
- SAM Audioで抽出した対象音声、除去音声をそれぞれ再生可能にする
- 音声は保存可能にする
- 動作環境はMacbook
- 可能な限り商用利用可能なライセンスのライブラリ・モデルで構築する
    - ライセンスがあやし場合は、ユーザーに相談すること
- testは不要だが、ソースコードをきちんと実装すること
- 環境はuvで管理している
